# PaddimiGlobal

Nigerian-focused web application for requesting legal affidavits and newspaper publications online — no account required.

## Tech Stack

- **React 19** + TypeScript
- **Vite** for build tooling
- **Tailwind CSS v4** for styling
- **React Router** for navigation
- **Lucide React** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page with service overview |
| `/affidavit` | 4-step affidavit request wizard |
| `/newspaper` | 4-step newspaper publication wizard |
| `/track` | Track request status by redemption code |

## Demo Tracking Codes

- `PG7K2M9X4N` — Affidavit of Name Change (processing)
- `NH3R8W1Q5T` — Loss of Document Publication (ready)
